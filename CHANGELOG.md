# Changelog

All notable changes to `lite-llm-inference` are documented in this file.

## [0.1.0] - 2026-03-01
### Added
- TierSet selection engine and budget solver.
- Deterministic token routing pipeline and expert packing/dispatch model.
- Dynamic prefetch planning and KV-cache architecture.
- Streaming runtime with backpressure/heartbeat/replay-prefix behavior.
- Cost-adaptive routing, telemetry collection, and tenant-isolation controls.
- Determinism and regression tests for latency/throughput and isolation behavior.
